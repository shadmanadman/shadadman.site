## may-12-2025-sportsgamex-KMP-Integration

I got a new and interesting responsibility. Implementing KMP modules to be shared between ios and android source code. This modules will hold common logics that can be shared between both platform.

After a couple of meetings with the team leads, I explained why these modules could be a huge help and how we planned to use them. Both the Android and iOS teams were happy about it. These modules can lift a weight off their shoulders. 

We had a couple of features that can be implemented with this modules:

- Google sign in 
- Analyzer platforms, Firebase and mixpannel
- TradingView charts

The implementation of these features was repetitive work on both mobile platforms. Now, instead of having the team develop them again, they can focus on new features.

I became responsible not only for developing these features but also for implementing their CI/CD pipelines and injecting them into both of our mobile platforms.
